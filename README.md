# Login Page Project

## Overview
This is a simple, responsive login page created using HTML and CSS.

## Features
- Clean and modern design
- Responsive layout
- Input validation
- Mobile-friendly

## Technologies Used
- HTML5
- CSS3

## How to Use
1. Download the `index.html` file
2. Open the file in any web browser

## Responsive Design
The login form adapts to different screen sizes, providing a seamless experience on desktop and mobile devices.

## Customization
You can easily modify the CSS in the `<style>` section to change colors, fonts, and layout.

## Installation
No installation required. Just open the HTML file in a browser.
